
# TIPURI DE DATE PRIMITIVE SI REFERINTA

## INTRODUCERE
#
- ### In JavaScript avem obiecte peste tot
- ### Cam orice este un obiect; sau poate fi apelat prin intermediul unui obiect
#
- ### Pana si functiile sunt tot obiecte
#
- ### Putem crea obiecte oricand, si putem adauga sau sterge proprietati de cate ori este nevoie
#
> ### Cele doua tipuri de date, primitive si referintele, se comporta diferit

---

#
- ### Tipurile primitive sunt stocate ca tipuri de data simple

- ### Referintele sunt stocate ca obiecte, 
- ### Variabilele sunt adrese catre locatii de memorie (heap)

---

> ### JavaScript ne permite sa folosim tipurile primitive de date la fel ca obiectele
#
### JavaScript introduce conceptul de obiect (wrapper) pentru variabile

- ### Tipurile de date primitive sunt stocate direct in acest obiect
#
### Pentru tipurile referinta se stocheaza doar pointer-ul catre zona de memorie unde se afla obiectul
